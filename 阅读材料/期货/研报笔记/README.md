### 趋势策略
#### 一、日间策略
**1.ATR通道**  
TR(True Range) = Max(  
	当前交易日最高价 - 当前交易日最低价，
	abs(当前交易日最高价 - 前一交易日收盘价），
	abs(当前交易日最低价 - 前一交易日收盘价)
)

参数：通道带宽k, 观察期长度N
